# NexaFlow

Business management dashboard with email automation via n8n.

## Stack
- **Frontend** — Next.js 14 + Tailwind CSS
- **Backend** — Node.js + Express.js
- **Database** — PostgreSQL + Prisma
- **Auth** — JWT + bcrypt
- **Automation** — n8n Webhook

## Get Started

```bash
git clone https://github.com/Omar1Ach/NexaFlow.git
cd nexaflow
npm install
npm run dev
```
